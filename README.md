# Drehabwen

I build rehabilitation AI systems from technical primitives to clinical-facing products.

The current work centers on **DeepRehab / Drehab**: motion assessment, clinical voice structuring, medical agent workflows, and practical products for rehabilitation screening, pediatric spine-health follow-up, and medical research.

## Portfolio Layers

```text
technical labs     -> motion metrics, clinical voice, medical agent guardrails
product systems    -> screening apps, family follow-up, research assistants
workflow tooling   -> Codex skills and AI production pipelines
```

## Direction

The long-term direction is to make rehabilitation and medical AI more operational:

```text
screening data
-> technical evidence
-> clinical reasoning support
-> family-side execution
-> research evidence
-> reproducible AI workflows
```

The goal is not to replace clinicians, but to build tools that help clinical teams and families capture better information, follow through on plans, and communicate more clearly.
